class ClipGenerator(builtins.object)
    ClipGenerator(lines, start_times, end_times, filename, audio_filepath, font_file, fonts_dir, directory)
 
this class is given
    - lines: a list of words (str), each line of words to be displayed together
    - start_times: a list of numbers (str), the start time of each line
    - end_times: a list of numbers (str), the end time of each line
    - filename: all results will be saved with this name
    - audio_filepath: audio filepath to accompany words, can be empty
    - font_file: font filepath for the words on each line
    - directory: directory where results and temporary files are saved
and generates a clip with text that matches the audio clip (if given)
